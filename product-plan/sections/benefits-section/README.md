# Benefits Section

## Overview

Section showcasing why clients should choose EVAZ-MS. Features 4 benefit cards addressing pain points, followed by a team photo carousel.

## User Flows

1. **Read Benefits** → User scrolls → sees 4 benefit cards
2. **View Team** → User sees team photo carousel
3. **Read FAQs** → User clicks FAQ items → expands answers

## Benefits Content

1. **Професійний результат** — Shield icon — Addresses fear of penalties
2. **Спокій і впевненість** — Check icon — Addresses uncertainty
3. **Свобода зосередитись** — Rocket icon — Addresses time drain
4. **Стабільна команда** — Users icon — Addresses trust concerns

## Components Provided

- `BenefitsSection.tsx` — Main section layout
- `BenefitCard.tsx` — Individual benefit card
- `TeamCarousel.tsx` — Team photo carousel

## Callback Props

| Callback | Description |
|----------|-------------|
| `onCtaClick` | CTA button clicked |
| `onFaqToggle` | FAQ item toggled with (id) |
